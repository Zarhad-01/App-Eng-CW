Help Section

Saving and loading

Loading is done automatically when the webpage loads.

Saving a file is a simple case of clicking on the save button or by using the inbuilt keyboard shortcut (see below).

Deleting

Deleting will also delete any child nodes.

Keyboard Shortcuts

  • Enter - Create new line.
  • Ctrl + Enter - Create new Indented line.
  • Alt + Enter - Create new line at the bottom.
  • Ctrl + S - Save file
  • Ctrl + B - Make text bold
  • Ctrl + I - Italicize text
  • Ctrl + P - Print page
  • Alt + + - Expand
  • Alt + - - Collapse
  • Ctrl + ↑ - Move selected line up.
  • Ctrl + ↓ - Move selected line down
  • Ctrl + ← - Indent Left
  • Ctrl + → - Indent Right
